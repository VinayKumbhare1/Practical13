Title: Responsive Mini Website

Description:
This project creates a responsive mini website using Bootstrap components such as Form, Modal, Dropdown, and Breadcrumb.

Features:

Responsive layout
Dropdown menu
User input form
Modal popup
Breadcrumb navigation

How to Run:

Open the HTML file in a browser
Click dropdown and modal button to test features
Fill the form for demo

Requirements:

Internet connection (Bootstrap CDN)
Browser
